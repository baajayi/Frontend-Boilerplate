Object-Oriented Programming in JavaScript (OOP)

programming in JavaScript is similar to OOP in other programming languages. It involves breaking down codes into objects that have methods and properties. The three main concepts in OOP are Encapsulation, Inheritance and Polymorphism. Encapsulation makes a method available for implementation without seeing the inner workings of the method. Polymorphism is when different objects implement the same methods in their own ways. Inheritance is when the methods and properties of an already existing object are implemented by another object in its own specific way. Classes defines the blueprint for an object. When an object is created as an instance of a class, it inherits all the properties and methods of that class. A constructor function defines the properties and methods of an object. Object, Array and Function are JavaScript built-in constructor functions.

Object Method .this
